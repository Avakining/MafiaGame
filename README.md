# Mafia

This is a Java-based GM's utility for running a game of Mafia. It's fully configurable: it supports any number of players, factions, role categories, and roles.

<details>
<summary>How To Use</summary>
To run, download the [latest release](https://github.com/Avakining/MafiaGame/releases/latest) of `MafiaGame.jar` and put a properly configured `players.txt` and `factions_roles.toml` in the same directory, as shown below.
